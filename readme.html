MsgText Package

This package contains the MsgText application. MsgText is used to convert .msg files to .txt files and extract all attachments.

For long-term preservation of mail the resulting attachment files will have to be converted further.

For extracting .msg files from a message store, the Enter AG utility MsgExport can be used.

The command-line application MsgText.exe can be found in the bin folder. MsgText -h displays a short usage notice.

An interactive user interface for the command-line application can also be found in the bin folder. Its name is mt.js. Normally double clicking it with the mouse will start it. If that fails, call it from the command-line using wscript: wscript bin/mt.js.

Please consult the manual for more information about MsgText.
